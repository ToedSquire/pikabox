# pikabox
The better operating system.
## About
Pikabox is a highly capable and customisable operating system that supports a wide range of use cases, thanks to its Ubuntu and Mint Linux base. To give it a user-friendly, responsive look, the Budgie desktop environment is used, but it can be swapped for any desktop environment that uses lightDM.
## Installation
To install Pikabox, the only current option is to use a USB stick and download the .iso onto it.
